## Style Guidelines for All Images

### Color Palette
- **Primary Purple**: #4E49FC (buttons, headers)
- **Secondary Purple**: #6366F1 (chat bubbles, accents)
- **Blue**: #3B82F6 (progress bars, links)
- **Background**: #FFFFFF (white)
- **Light Gray**: #F3F4F6 (input fields, AI bubbles)
- **Text**: #111827 (dark gray)

### Design Elements
- Rounded corners: 12-20px
- Subtle shadows: soft, light gray
- Modern iOS 17 design language
- Clean, minimalist aesthetic
- Professional mobile app UI
- High contrast for readability

### Technical Specifications
- Device: iPhone 15 Pro Max (or latest iPhone)
- Orientation: Portrait
- Resolution: 8K or highest available
- Style: Realistic app screenshot
- Lighting: Soft, natural
- Quality: Professional, App Store ready

## Usage Instructions

1. **For Midjourney:**
   - Use the full prompts as-is
   - Add `--ar 9:16` for portrait aspect ratio
   - Add `--v 6` or latest version
   - Add `--style raw` for more realistic results

2. **For DALL-E 3:**
   - Use the shorter alternative prompts
   - Specify "iPhone app screenshot" clearly
   - Request "high quality, professional"

3. **For Stable Diffusion:**
   - Use the full prompts
   - Add negative prompts: "blurry, low quality, distorted text, watermark"
   - Use realistic model checkpoints

4. **Post-Processing:**
   - Ensure text is readable
   - Verify colors match brand guidelines
   - Add device frame if needed
   - Optimize for App Store requirements (various sizes)

---

## App Store Requirements

- **Screenshot Sizes Needed:**
  - 6.7" Display (iPhone 15 Pro Max): 1290 x 2796 pixels
  - 6.5" Display (iPhone 11 Pro Max): 1242 x 2688 pixels
  - 5.5" Display (iPhone 8 Plus): 1242 x 2208 pixels

- **Best Practices:**
  - Show actual app functionality
  - Use real content (not placeholder text)
  - Highlight key features
  - Maintain consistent style across all screenshots
