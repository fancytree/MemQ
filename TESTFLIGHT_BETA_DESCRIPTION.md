# TestFlight Beta Description

## Version 1.0.0 (Build 1) - Beta Testing

### App Overview
MemQ is an intelligent learning assistant that combines AI technology with scientific memory methods to help you learn and remember more effectively.

### Key Features
- 📚 **Course Management**: Create personalized learning courses and add terms with definitions
- 🤖 **AI Assistant**: AI-powered question generation, concept explanations, and learning suggestions
- 📄 **PDF Import**: Upload PDF documents and automatically extract learning content
- 🎯 **Flashcard Learning**: Science-based spaced repetition memory method
- 📊 **Progress Tracking**: Real-time learning progress and mastery tracking
- 🔔 **Study Reminders**: Customizable reminder notifications to maintain learning habits

### Testing Focus
